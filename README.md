A lightweight script to track your Hackatime stats for any YSWS

Made for Hack Club Horizons Crux  


## INSTALLATION:

1. Download `Blinky` from releases
2. Right click the file
3. Select "Run in terminal"
4. Done!

## USAGE:

Just follow the prompts, and it'll give you lots of data, including:

- The required time you set
- The time you have
- The amount of time you need to get
- An estimation of how long you'd need to code daily to meet your goal
- How much you've completed, in a percentage
